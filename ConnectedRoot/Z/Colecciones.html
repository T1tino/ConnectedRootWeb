Sistema de Monitoreo de Huertos Urbanos

Colecciones Base de Datos

Colección Huertos
Descripción: Representa los huertos urbanos registrados en el sistema para monitoreo y gestión integral.
Nombre del campo Descripción
_id Identificador único del huerto
nombreHuerto Nombre descriptivo del huerto urbano
ubicacion Dirección física completa del huerto
responsableId Referencia al usuario responsable
fechaRegistro Fecha de alta en el sistema
estado Estado del huerto (activo/inactivo)
Huertos
_id: ObjectId
nombreHuerto: string
ubicacion: string
responsableId: ObjectId
fechaRegistro: date
estado: string
Colección Zonas
Descripción: Representa las zonas específicas dentro de cada huerto para organización y monitoreo detallado.
Nombre del campo Descripción
_id Identificador único de la zona
huertoId Referencia al huerto al que pertenece
nombreZona Nombre identificativo de la zona
descripcion Descripción detallada de la zona
coordenadas Objeto con latitud y longitud GPS
tipoZona Tipo o función de la zona
estado Estado operativo (activo/inactivo)
Zonas
_id: ObjectId
huertoId: ObjectId
nombreZona: string
descripcion: string
coordenadas: object
tipoZona: string
estado: string
Colección Sensores
Descripción: Representa los dispositivos IoT instalados en las zonas para monitoreo ambiental continuo.
Nombre del campo Descripción
_id Identificador único del sensor
zonaId Referencia a la zona donde está instalado
tipo Tipo de medición (temperatura/humedad)
modelo Modelo específico del sensor
estado Estado de la zona (activo/inactivo)
fechaInstalacion Fecha de instalación del sensor
descripcion Observaciones adicionales del sensor
Sensores
_id: ObjectId
zonaId: ObjectId
tipo: string
modelo: string
estado: string
fechaInstalacion: date
descripcion: string
Colección Lecturas
Descripción: Almacena las mediciones en tiempo real capturadas por los sensores IoT del sistema.
Nombre del campo Descripción
_id Identificador único de la lectura
sensorId Referencia al sensor que generó la lectura
fechaHora Timestamp exacto de la medición
tipo Tipo de dato registrado
valor Valor numérico medido
unidad Unidad de medida (°C, %)
Lecturas
_id: ObjectId
sensorId: ObjectId
fechaHora: date
tipo: string
valor: number
unidad: string
Colección Alertas
Descripción: Registra las alertas automáticas generadas cuando se detectan condiciones críticas en el sistema.
Nombre del campo Descripción
_id Identificador único de la alerta
tipo Tipo específico de alerta generada
zonaId Referencia a la zona afectada
valorRegistrado Valor que desencadenó la alerta
umbralMinimo Límite inferior configurado
umbralMaximo Límite superior configurado
valorUmbralViolado Valor específico del umbral violado
mensaje Descripción detallada de la alerta
fechaHora Momento exacto de generación
estado Estado actual (pendiente/resuelta)
enviada Indicador de notificación enviada
Alertas
_id: ObjectId
tipo: string
zonaId: ObjectId
valorRegistrado: number
umbralMinimo: number
umbralMaximo: number
valorUmbralViolado: number
mensaje: string
fechaHora: date
estado: string
enviada: boolean
Colección Usuarios
Descripción: Almacena la información de los usuarios que gestionan y monitorean los huertos urbanos.
Nombre del campo Descripción
_id Identificador único del usuario
nombre Nombre(s) de pila del usuario
primerApellido Primer apellido del usuario
segundoApellido Segundo apellido del usuario
correo Dirección de correo electrónico
contraseña Contraseña de acceso
rol Rol en el sistema (administrador/operador/agricultor)
huertosAsignadosIds Array de referencias a huertos asignados
estado Estado de habilitación del usuario
Usuarios
_id: ObjectId
nombre: string
primerApellido: string
segundoApellido: string
correo: string
contraseña: string
rol: string
huertosAsignadosIds: array
estado: boolean
